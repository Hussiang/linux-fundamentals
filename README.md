# Linux Fundamentals for DevOps

A practical, hands-on guide to Linux fundamentals with a focus on real-world DevOps and Cloud Operations use cases. This repository documents my learning journey through explanations, examples, troubleshooting scenarios, and automation scripts.

---

## Why this repository?

Linux is the foundation of modern cloud infrastructure and DevOps. Rather than memorizing commands, this repository focuses on understanding how Linux works and how its concepts are applied in production environments.

The goal is to build practical knowledge that can be applied while working with cloud platforms, containers, CI/CD pipelines, and infrastructure automation.

---

## Topics Covered

- Linux Basics
- Filesystem Hierarchy
- File & Directory Management
- Users and Groups
- Permissions and Ownership
- Processes
- System Services (systemd)
- Package Management
- Networking
- Storage
- Log Management
- Shell Scripting
- Security Basics
- Troubleshooting
